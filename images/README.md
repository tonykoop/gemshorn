# Images

This folder is reserved for real shop photos and finished instrument photos.

Until prototypes exist, the repo uses SVG drawings in `drawings/` and `site/assets/drawings/` as visual placeholders. Use `photo-shotlist.md` for the required photo sequence and save future files here with stable names such as:

- `hero-family-lineup.jpg`
- `horn-001-raw-blanks.jpg`
- `cer-002-plaster-mold-open.jpg`
- `cnc-002-routed-cavity-halves.jpg`
- `val-001-closed-note-tuner.jpg`
